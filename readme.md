## Laravel 5.5 & Vue Router + JWT(Auth)
Vue SPA with Laravel + JWT for auth example template project setup  

## Installation

<pre>
    1. npm install
    2. composer install  
</pre>

## Start server

<pre>
    1. php artisan serv --port [portNumber] #default 8000
    2. npm run watch #live reload after change file  
</pre>
